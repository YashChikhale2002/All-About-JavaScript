• (>, <,>=, <=,===, !==) These are Comparison Operators.
  • Instead of == we use === To avoid the conversion behavior.
  • Instead of != we use !== To avoid the conversion behavior.
  • Comparison Operators have a lower priority then Math.
  • (77, ||, !) These are Logical Operators.
  • Logical have a lower priority then Math & Comparison Operators.
  • Falsy Values : false, 0, "", NaN, Undefined, Null.
  • Truthy Values : Any value Not on this is truthy.
  • Ternary Operator: Condition ? ifCondTrue(if) : ifCondFalse(else)
  • Advantage of this Operator is that we can save it in the variable.
