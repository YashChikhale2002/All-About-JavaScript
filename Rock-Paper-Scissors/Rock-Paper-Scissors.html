Rock Paper Scissors

Workflow of Rock Paper Scissors

  • First of all we create 3 buttons with onclick event.
  • Then we create a function which generates a random number.
  • Based on the random numbers it defines rock, paper, scissors.
  • Then we create function for player move.
  • Then we compare both functions using if-else. to choses winner.
  • We use return statement to make block scope a global scope var.
  • Then Show the final result in alert.
